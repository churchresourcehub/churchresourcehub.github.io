# Doing Church Together — Church Resource Hub

A shared library of systems, processes, and working documents from churches. One page, one flat document vault behind it. Find a real example, download it, and adapt it for your congregation.

Started by Spanish Fort UMC. Built for everyone doing church together.

To contribute: email your file to churchresourcehub@outlook.com with your church name, ministry area, year, and a one-line description.
